# Application

This project contains a maven application 

## Prerequisites
- Java 11
- Apache Maven

## Development

Below is the structure of the generated project.

```
├── src
│   ├── main
│   │   ├── java
│   │   │   └── package
│   │   │       └── Application.java
│   │   └── resources
│   │       └── application.yml
│   └── test
│       └── java
```

- `Application.java`: main entry of the application

#### Building the project
```
mvn clean package
```
